Abstraction for Menial Things

Ever wanted that one small thing to be abstract? No..? Oh well, i did it anyway. Probably not, but i did some things.

Features:

  • Abstract JSON Data Resource Loader
  • Abstract JSON Config File Loader
  • General Utility Classes
  • Registration Helpers
  • Built-in Fabric ASM based Shield and Elytra Enchantment Targets
  • ImplementedInventory Interface from Fabric Wiki
  • Attempted and Likely Quilt Support
  • /libgenetics command to check mod status
  • Nullable-Ingredient Mixin
  • Continual Updates to JitPack using Github Commit Hashes
